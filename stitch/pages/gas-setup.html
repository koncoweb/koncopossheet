URL Web App (deployment)

Cara A (disarankan): buka Google Sheet Anda → Extensions → Apps Script → tempel semua file .gs dari folder stitch/gas/ (bukan Code.gs + modular sekaligus). Deploy Web App → salin URL ke sini.

Error “Spreadsheet tidak terhubung”? Artinya script tidak tahu sheet mana. Kalau project Apps Script dibuat di script.google.com: buka Apps Script → ikon gerigi (Project Settings) → Script properties → tambah SPREADSHEET_ID = ID dari URL sheet (bagian …/d/ID_INI/edit…) atau paste URL penuh. Simpan lalu Deploy → kelola deployment → Edit → Versi baru.
